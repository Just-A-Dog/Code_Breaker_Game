Code Breaker is a simple game I made in my off time.

-=RULES=-
  -GOAL: The goal of the game is to input the correct color sequence on the bottom row using the other rows as hints. 
  The player is able to cycle through each color by clicking on the proper square on the bottom row.

  -HINTS: Each row above the inputted row provides a 'guess' toward the answer. The symbols to the right of the row provide hints.
  The number of stars tells the player the number of colors in that row that are in the correct spot for the final answer.
  The number of dots tells the player how many colors are correct, but are in the wrong position.

There is only one answer that makes all of the hint rows make sense. 
When the player wants to submit their guess, they will click the 'check' button, and they will find out whether or not their color combination is correct.
